What is a podcast?

Podcasts are subscriptions to regularly produced multimedia (usually MP3) that can be automatically downloaded for later listening. It requires the use of podcast or "podcatching" software such as the popular iTunes (version 4.9 or newer), which is a free download for Windows or Mac. If you have an iPod, iTunes can add these MP3s to it automatically.

We currently maintain three podcasts, consisting of the latest bible teachings by Dennis McCallum, Gary DeLashmutt, and Jim Leffel. You can subscribe to any or all of them to obtain their teachings automatically as they become available.

Do I need to have an iPod?

No. You can listen to the teachings directly on your computer. Or you can create a playlist in iTunes and burn a cd that can be played anywhere.

How much does it cost to subscribe to these podcasts?

Nothing! We offer these subscriptions and MP3 downloads for free.

How do I get started?

  1. Install podcast software. We recommend downloading iTunes for Windows or Mac—for FREE!
  2. Click the "iTunes 1-Click" button above for the desired podcast if you're using iTunes. If you're using different podcast software, copy and paste the URL specified into your application.
